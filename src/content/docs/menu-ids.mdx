This is the complete reference of all menu IDs (`IDM_VS_*`) and command group IDs (`IDG_VS_*`) defined in the Visual Studio SDK. All IDs use the `guidSHLMainMenu` GUID: `{D309F791-903F-11D0-9EFC-00A0C911004F}`

---
